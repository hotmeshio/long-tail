Workflow orchestrator — `executeLT()` composes child workflows with automatic task tracking, provider data injection, and lifecycle hooks. It uses `startChild` (fire-and-forget) to spawn children and `waitFor` to receive result signals from the interceptor, making it resilient to any number of escalations, failures, and re-runs in the child workflow. `index.ts` is the single module, exporting `executeLT` and the `ExecuteLTOptions` interface.
